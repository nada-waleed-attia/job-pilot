# JobPilot - AI-Powered Job Application System

## 🎯 الوصف
JobPilot هو نظام ذكي لبحث وتقديم الوظائف يعمل بالذكاء الاصطناعي، يقوم بتحليل بيانات المستخدم وتقديمه على الوظائف المناسبة تلقائياً.

## 🚀 المميزات الرئيسية

### 🤖 نظام الذكاء الاصطناعي
- **تحليل بيانات المستخدم**: تحليل الخبرات، المهارات، والراتب المستهدف
- **تقديم تلقائي**: تقديم على الوظائف المناسبة تلقائياً
- **وظائف مخصصة**: بناءً على مجال المستخدم وموقعه المفضل
- **إشعارات فورية**: إشعارات عن كل تقديم جديد

### 👤 نظام المستخدم
- **تسجيل بيانات شامل**: معلومات شخصية، مهنية، وتقنية
- **عرض اسم المستخدم**: في جميع صفحات الموقع
- **تسجيل خروج آمن**: مسح البيانات وتسجيل الخروج
- **حفظ البيانات**: بشكل دائم في localStorage

### 📊 نظام الإشعارات
- **إشعارات تفاعلية**: عرض جميع التقديمات
- **تفاصيل الوظائف**: عرض كامل تفاصيل كل وظيفة
- **نوافذ منبسقة**: عرض الوظائف المقدمة عليها
- **تحديث تلقائي**: كل 20 ثانية

### 🎨 التصميم
- **تصميم عصري**: Bootstrap 5 + CSS3
- **رسوم متحركة**: AOS Animation
- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **واجهة سهلة**: تجربة مستخدم سلسة

## 🚀 كيف يعمل النظام

### 1. تسجيل المستخدم
- المستخدم يدخل بياناته في `ai-system.html`
- يتم تحليل البيانات وحفظها في localStorage
- يتم إنشاء أول تقديم على وظيفة

### 2. الصفحة الرئيسية
- يتم عرض اسم المستخدم في الهيدر
- يتم عرض الإشعارات للوظائف المقدمة
- يبدأ النظام الذكي في البحث عن وظائف جديدة

### 3. التقديم التلقائي
- كل 20 ثانية يحاول النظام التقديم (30% احتمال)
- يتم اختيار وظائف بناءً على بيانات المستخدم
- يتم حفظ التقديمات الجديدة في localStorage

### 4. الإشعارات
- عرض جميع التقديمات في جميع الصفحات
- عرض تفاصيل الوظائف عند الضغط على الإشعارات
- إمكانية عرض جميع الوظائف المقدمة عليها

## 🔧 التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone https://github.com/yourusername/JobPilot.git
cd JobPilot
```

### 2. تشغيل Frontend
- افتح `frontend/html/main-dual.html` في المتصفح
- أو استخدم أي خادم محلي (Live Server, XAMPP, etc.)

### 3. تشغيل Backend (اختياري)
```bash
cd backend
pip install -r requirements.txt
python main.py
```
