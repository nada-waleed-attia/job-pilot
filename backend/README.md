# JobPilot - منصة ذكية للبحث عن وظائف والتقديم عليها أوتوماتيك

## وصف المشروع

JobPilot هي منصة ذكية تساعد الباحثين عن عمل في العثور على وظائف مناسبة والتقديم عليها تلقائياً. النظام يجمع الوظائف من مصادر متعددة مثل LinkedIn و Indeed ويطابقها مع ملف المستخدم الشخصي.

## المميزات الرئيسية

### 🔐 إدارة المستخدمين
- تسجيل حساب جديد وتسجيل الدخول
- إدارة الملف الشخصي والسيرة الذاتية
- تحديد التفضيلات والمهارات

### 🔍 البحث الذكي عن الوظائف
- بحث متقدم في وظائف متعددة المصادر
- فلترة حسب النوع والموقع والراتب
- حساب نسبة المطابقة لكل وظيفة

### 🤖 التقديم التلقائي
- تقديم تلقائي على الوظائف المناسبة
- إنشاء رسائل تعريفية مخصصة
- متابعة حالة التقديمات

### 📊 لوحة المتابعة
- إحصائيات مفصلة عن التقديمات
- متابعة حالة الطلبات
- رسوم بيانية تفاعلية

## التقنيات المستخدمة

### Frontend
- **HTML5/CSS3/JavaScript** - الأساسيات
- **Bootstrap 5** - تصميم متجاوب
- **AOS (Animate On Scroll)** - انيميشن

### Backend
- **Python** - لغة البرمجة
- **FastAPI** - إطار العمل
- **SQLAlchemy** - ORM للقاعدة البيانات
- **JWT** - المصادقة
- **Pydantic** - التحقق من البيانات

### الأتمتة
- **Selenium** - أتمتة المتصفح
- **Playwright** - بديل أسرع
- **BeautifulSoup** - تحليل HTML

## طريقة التشغيل

### تشغيل الواجهة الأمامية
1. افتح مجلد `frontend`
2. افتح `index.html` في المتصفح
3. أو استخدم خادم محلي مثل Live Server

### تشغيل الواجهة الخلفية
1. ثبت Python 3.8+
2. انسخ `.env.example` إلى `.env` وعدّل القيم
3. ثبت المكتبات:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
4. شغّل الخادم:
   ```bash
   python main.py
   ```

## الـ API Endpoints

### المصادقة
- `POST /api/auth/register` - إنشاء حساب جديد
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/users/profile` - جلب الملف الشخصي
- `PUT /api/users/profile` - تحديث الملف الشخصي

### الوظائف
- `POST /api/jobs/search` - البحث عن وظائف
- `POST /api/jobs/{job_id}/apply` - التقديم على وظيفة

### لوحة التحكم
- `GET /api/dashboard/stats` - إحصائيات لوحة التحكم
- `GET /api/users/applications` - قائمة التقديمات

### الميزات المتقدمة
- `POST /api/auto-apply/toggle` - تفعيل التقديم التلقائي
- `POST /api/generate-cover-letter` - إنشاء رسالة تعريفية
